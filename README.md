# DecodeLabs AI Engineering Internship — Task Submissions

**Batch 2026 | Industrial Training Kit**

This repository contains my completed tasks for the DecodeLabs AI
Engineering Internship. Each task lives in its own folder with its own
code, dataset (where applicable), and README explaining how it works.

## 📁 Tasks

### [Task 1: Rule-Based AI Chatbot](./Task-1-Chatbot)
A chatbot that responds to predefined user inputs using dictionary-based
if-else logic, running in a continuous loop. Covers control flow,
input sanitization, and fallback handling.

**Skills:** Control flow, decision-making logic, basic AI concepts

---

### [Task 2: Data Classification Using AI](./Task-2-Data-Classification)
A supervised learning model (K-Nearest Neighbors) trained to classify iris
flowers into 3 species based on their measurements. Covers the full
ML pipeline: train/test split, feature scaling, model training, and
evaluation with a confusion matrix and F1-score.

**Skills:** Data handling, supervised learning basics, model training

---

### [Task 3: AI Recommendation Logic — Tech Stack Recommender](./Task-3-Recommender)
A content-based recommendation engine that matches a user's skills to the
most relevant tech career role, using TF-IDF vectorization and Cosine
Similarity — the same core technique behind real-world recommendation
systems like Netflix.

**Skills:** Logic building, pattern matching, recommendation concepts

## 🛠️ Tech Stack

- Python 3
- scikit-learn
- matplotlib, numpy

## 🚀 How to Run Any Task

Each task folder is self-contained. Navigate into it and follow its own
README, generally:

```bash
pip install -r requirements.txt
python <script_name>.py
```
